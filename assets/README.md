# Assets Folder

Place your sound file here!

## Instructions:

1. Add your sound file (e.g., `boop.mp3`, `boop.wav`, or `boop.m4a`) to this `assets/` folder
2. Make sure the filename matches what's in `pubspec.yaml` (currently set to `boop.mp3`)
3. If you use a different filename or format, update the `assets:` section in `pubspec.yaml`
4. Supported formats: MP3, WAV, M4A, OGG

## Supported Audio Formats:
- MP3 (recommended for web compatibility)
- WAV
- M4A
- OGG

After adding your file, run:
```bash
flutter pub get
flutter run -d chrome
```
